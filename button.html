Opcjonalne atrybuty:
  • autofocus - ustawia domyslnie focus na przycisk (zaznacza go)
  • disabled - wyłącza przycisk
  • form - z wartością w formie id formularza do ktorego sie odnosi
  • formenctype - okresla jak maja byc zakodowane dane
  • formmethod - okresla w jaki sposob maja byc wyslane dane formularza, tylko dla type="submit"
    • get
    • post
  • formnovalidate - okresla ze dane nie maja byc sprawdzane przed wysylka, tylko dla type="submit"
  • formtarget - okresla jak ma zostac wyswietlona odpowiedz po przeslaniu formularza, przyjmuje wartosci
    • _blank
    • _self
    • _parent
    • _top
  • name - okresla nazwe przycisku
  • type - okresla typ przyciusku
    • button
    • reset
    • submit
  • value - wartosc poczatkowa przycisku.
